# Test de Perfil Inversor

## Cómo publicar en Vercel (paso a paso)

### 1. Crear cuenta en GitHub
- Entrá a https://github.com y creá una cuenta gratuita si no tenés

### 2. Subir el proyecto a GitHub
- En GitHub, hacé click en **"New repository"**
- Nombre: `test-inversor` (o el que quieras)
- Dejalo en **Public**
- Click en **"Create repository"**
- Subí todos los archivos de esta carpeta (arrastrándolos o con el botón "uploading an existing file")

### 3. Publicar en Vercel
- Entrá a https://vercel.com y creá una cuenta con tu GitHub
- Click en **"Add New Project"**
- Seleccioná el repositorio `test-inversor`
- Vercel detecta automáticamente que es React
- Click en **"Deploy"**
- En 2 minutos tenés una URL pública tipo: `test-inversor.vercel.app`

### 4. Agregar a Beacons
- En tu Beacons, añadí un bloque **"Embed"**
- Pegá la URL de Vercel
- Guardá y listo

---
Ante cualquier duda, consultá a tu asesor técnico o escribí al soporte de Vercel.
